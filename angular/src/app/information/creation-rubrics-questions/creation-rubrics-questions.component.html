How to create a rubrics or question template for your assessment?

Create rubrics!

There are two ways to create a rubrics.

Upload Template

Suggested!

The excel template can be downloaded here.

  1. Just fill in the excel template with your custom rubrics. (Red color represents uneditable)
  2. Upload the rubrics to the system. That's it!
Manually Input

You will need to input your rubrics into the rubrics table.

  1. You can edit the cell by double clicking a row.
  2. Modify the details for each criterion.
  3. You can add/delete a column by clicking the or beside the first row (mark ranges).
  4. You can add a row by clicking the add row button beneath the table.
  5. You can delete a row by clicking the at the first column of the row.

Create question!

There are three ways to create a question template.

Upload Template

Suggested!

The excel template can be downloaded here.

  1. Just fill in the excel template with your custom questions. (Red color represents uneditable)
  2. Upload the rubrics to the system. That's it!
Upload Moodle-Exported Template

Suggested!

  1. Export the question in .csv format from your Moodle Question Bank.
  2. Upload the exported file in .csv format to the system.
Manually Input

You will need to input your question into the table.

  1. Click the + Add Question to add a new question.
  2. Input the question details.
  3. Click save to save the question.
  4. To delete the question, click the beside the edit button.
